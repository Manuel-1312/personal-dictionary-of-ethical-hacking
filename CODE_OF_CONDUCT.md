# Código de Conducta

**Autor:** Manuel Sánchez Brito

Este repositorio es un espacio de aprendizaje ético. Al contribuir o usar estos recursos, acepta:

1. **Actuar con autorización**: Solo ejecuta herramientas sobre sistemas con permiso explícito.
2. **Compartir conocimiento responsablemente**: Documenta fuentes, menciona licencias y evita publicar exploits dañinos.
3. **Mantener la privacidad**: No publiques credenciales ni datos sensibles reales.
4. **Corregir y reportar**: Si detectas errores o herramientas obsoletas, abre un issue o PR con contexto.

Respeta los principios de la comunidad y ayuda a otros a aprender en un entorno seguro.
