# Respuesta a Incidentes

Aquí anoto cada respuesta: detección, contención, erradicación y reporte. Cuando un sensor dispara, creo un archivo `CASE-YYYY-MM-DD.md` y lo enlazo desde este README. Incluyo qué alertas se dispararon (Zeek, Suricata, Wazuh), qué acciones tomé y qué artefactos (logs, dumps) generé.
