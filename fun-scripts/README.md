# Fun Scripts 🎉

No todo en un laboratorio tiene que ser serio. Esta carpeta reúne scripts pequeños, curiosos o directamente juguetones que sirven para demos, documentación, práctica o simplemente para darle un poco de personalidad al repositorio.

## Enfoque
- scripts cortos
- fáciles de ejecutar
- mezcla de utilidad ligera + humor + estética terminal
- pensados para labs, notas, demos o entrenamiento

## Scripts incluidos
- `ascii_banner_maker.py`
- `lab_quote_generator.py`
- `fake_breach_report.py`
- `terminal_glitch_text.py`
- `random_codenames.py`
- `phishing_subject_ideas.py`
- `password_wordlist_mixer.py`
- `red_team_excuse_generator.py`
- `log_noise_generator.py`
- `retro_matrix_printer.py`

## Recomendaciones
- Mantén esta carpeta ligera y divertida; si un script deja de ser pequeño o se vuelve muy útil, quizá merece vivir en `python/`.
- Si un script genera salidas reutilizables, guarda ejemplos en una subcarpeta `examples/` más adelante.
