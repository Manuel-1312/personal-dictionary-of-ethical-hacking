# Toolkit - Fun Scripts 🎈

| Script | Qué hace | Comando de ejemplo | Notas |
| --- | --- | --- | --- |
| `ascii_banner_maker.py` | Genera banners ASCII | `python fun-scripts/ascii_banner_maker.py --text "LAB 01"` | Bueno para notas o intros rápidas. |
| `lab_quote_generator.py` | Saca frases aleatorias de laboratorio | `python fun-scripts/lab_quote_generator.py` | Humor ligero con tono hacker/seguridad. |
| `fake_breach_report.py` | Genera un informe ficticio de incidente | `python fun-scripts/fake_breach_report.py --company Acme --output report.md` | Útil para practicar reporting. |
| `terminal_glitch_text.py` | Convierte texto en estilo glitch | `python fun-scripts/terminal_glitch_text.py --text "ACCESS GRANTED"` | Puro postureo terminalero. |
| `random_codenames.py` | Genera nombres de operación | `python fun-scripts/random_codenames.py --count 5` | Sirve para labs o casos. |
| `phishing_subject_ideas.py` | Genera asuntos ficticios de awareness | `python fun-scripts/phishing_subject_ideas.py --count 10` | Solo para simulaciones internas. |
| `password_wordlist_mixer.py` | Crea mini wordlists mezclando piezas | `python fun-scripts/password_wordlist_mixer.py --base manu lab cyber --years 2024 2025` | Útil para labs controlados. |
| `red_team_excuse_generator.py` | Genera excusas absurdas | `python fun-scripts/red_team_excuse_generator.py` | Humor interno. |
| `log_noise_generator.py` | Genera logs falsos para practicar | `python fun-scripts/log_noise_generator.py --count 50 --output fake.log` | Divertido y útil. |
| `retro_matrix_printer.py` | Imprime una animación tipo matrix | `python fun-scripts/retro_matrix_printer.py --seconds 5` | Para demos y estética. |
