# PowerShell Tools

Esta carpeta reúne utilidades PowerShell pensadas para Windows, inventario, reporting, red local y alguna que otra pieza visual para que no todo sea gris administrativo.

## Subcategorías
- `system/` → procesos, servicios, tareas, arranque y estado general del sistema.
- `files/` → inventario de carpetas, archivos grandes, extensiones y orden ligero.
- `network/` → instantáneas de red local, conexiones y comprobaciones básicas.
- `reporting/` → conversión de datos a formatos más legibles y plantillas rápidas.
- `fun/` → pequeños scripts visuales o simpáticos para consola y demos.

## Filosofía
- scripts sencillos
- cero o muy pocas dependencias extra
- útiles de verdad en Windows
- con outputs fáciles de guardar en CSV, JSON o Markdown
