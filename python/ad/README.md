# Python Active Directory Helpers

Esta subcarpeta contiene utilidades para ordenar hallazgos y notas de laboratorios Active Directory. La idea es ayudar a documentar mejor usuarios, permisos, hosts y mitigaciones sin depender siempre de notas manuales.

## Scripts actuales
- `ad_notes_builder.py` → genera una nota Markdown estructurada a partir de datos básicos de un caso AD.

## Recomendaciones
- Usa estos helpers solo en laboratorios o dominios autorizados.
- Mantén un formato de nota consistente para que luego sea fácil comparar casos.
