# Python Recon Helpers

Esta subcarpeta agrupa scripts para comparar, limpiar y resumir hallazgos de reconocimiento. Son utilidades pequeñas pensadas para OSINT, subdominios, inventario y seguimiento de cambios entre sesiones.

## Scripts actuales
- `subdomain_diff.py` → compara dos listas de subdominios y resalta altas, bajas y elementos comunes.

## Recomendaciones
- Guarda entradas y salidas con fecha para poder ver evolución real entre sesiones.
- Si el helper genera un resumen útil, enlázalo desde `recon/` o `osint/`.
